Localization Usage Guidelines

1. Always use translation keys from localization files (e.g., localizations/en/translation.json).
2. Do not use hardcoded strings directly in the UI.
3. Before adding a new key, check existing translation files to see if the same or a similar text already exists.
4. Reuse existing keys whenever possible to maintain consistency.
5. If a new string is required, add it to all supported language files with a clear and meaningful key name.
