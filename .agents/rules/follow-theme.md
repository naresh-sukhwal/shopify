Theme Usage Guidelines

1. Always use @themeStore and themeUtils for handling theme-related values.
2. Do not use arbitrary or hardcoded colors.
3. Before adding a new color, check themeUtils to see if an exact or closely matching color already exists.
4. Reuse existing colors whenever possible to maintain consistency.
5. If a new color is necessary, add it to the theme with a clear and meaningful name.
